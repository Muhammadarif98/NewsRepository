##Project: NewsRepository
Android News App: An application for viewing current news using a third-party API, displaying a list of news articles and allowing users to read full texts
Stack:
Kotlin
MVVM
Clean Architecture
Dagger Hilt
Retrofit
Coroutines
Glide
Material Design
REST API
